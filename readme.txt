== Description ==

Hero – Interactive WordPress Portfolio, best fit for creative and corporate  websites. You can display portfolio as three, two and one column. Additionally you can also display portfolio by category. Hero also supports single page, multiple images or video for each portfolio item. You can set up excerpt length for each of display type (one, two and three columns).

== Installation ==

1. Upload the plugin: Plugins > Add new > Upload.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to main menu "Hero Portfolio" > Add new (add portfolio items).
4. Go to main menu "Hero Portfolio" > "Hero Settings" (edit settings and see shortcodes).
5. Add shortcodes within your pages.
